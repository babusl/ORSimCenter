HapticLung Firmware
hapticlung_firmware_mk5.ino
ESP32-C3 firmware for the HapticLung Mk.5 physical resistance device. Changes APL valve position in real time based on ventilator pathology. Uses WiFiManager for setup — connect to "HapticLung-Setup" hotspot on first boot, enter your WiFi credentials once. Flash via Arduino IDE or esptool-js in Chrome.
ESP32-C3 WebSocket 2.4GHz only

APL Valve Servo Coupler
apl_servo_coupler.scad
OpenSCAD file for the 3D-printable adapter that couples an SG90 servo to the APL valve adjustment knob. Press-fit sleeve with internal grip nubs that grab the knurled knob surface. Set KNOB_OD to your valve's diameter (measure with calipers). Export STL, print in PLA at 0.2mm layers, 40% infill. Tune fit in 0.2mm increments.
OpenSCAD SG90 Servo PLA · 30 min print

Quick Start — Supervised Session
STEP 01
Start monitor
Open medsim-10.html on bedside screen. Note the room code.
STEP 02
Connect ventilator
Open medsim-vent-5.html on a second screen or tablet.
STEP 03
Instructor connects
Open controller-16.html on instructor phone, enter room code.
STEP 04
Learners connect
Each learner opens participant-controller.html and enters the same room code.
SELF-DIRECTED
Solo practice
